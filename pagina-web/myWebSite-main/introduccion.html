Introducción al Proyecto

Motivación

Motivados por la entrega del trabajo final de curso de la materia Introducción a computación en la nube con AWS, se presentó al grupo un data set con metales y sus diferentes propiedades físicas y mecánicas, con el fin de resolver un problema aplicando las herramientas brindadas por los docentes del curso.

Data Set

Dentro del data set se cuenta con valores de Identificación, resistencia a la tracción, resistencia de fluencia, módulo de Young, módulo de corte, coeficiente de Poisson y densidad.

Esta información es muy valiosa a la hora de seleccionar materiales para el diseño de máquinas y estructuras, ya que se debe asegurar que cumpla su función sin dañarse y optimizando recursos como peso y costo.

Desarrollo de la API

Para facilitar el proceso de selección del mejor metal para la aplicación deseada por el usuario, se desarrolló una API que con los inputs correspondientes al tipo de proyecto deseado devuelve una lista con los 10 mejores metales para esa aplicación.

Nuestros inputs serán rangos de E, rangos de Rho, y la API se encarga de devolver una lista con los 10 metales en esos rangos optimizando la relación de tensión de fluencia (Sy) sobre la tensión de ruptura (A5), con el criterio de que cuanto más cercano a 1 sea dicho índice, peor va a ser el material para la aplicación deseada en este caso, ya que queremos que la tensión de ruptura esté lo más lejos posible del límite elástico.

Explicación del Índice Sy/A5

Los valores que en el data set están denominados por Sy, Su y A5 son los correspondientes a los puntos B, D y E respectivamente.

La idea de minimizar el índice Sy/A5 hace referencia a la relación de esfuerzos para llegar a la ruptura del material sobre el esfuerzo con el que el material abandona la zona plástica. Dicho índice, cuanto menor sea, mayor va a ser el esfuerzo para llegar a la ruptura luego de pasar el límite elástico, por lo que el material será más difícil de romper.
